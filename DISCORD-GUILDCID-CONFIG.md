## 🔍 WIE FINDEST DU DEINE GUILD ID?

### Methode 1: Discord Developer Mode (Empfohlen)
1. Discord öffnen
2. Einstellungen → Erweitert
3. "Entwicklermodus" aktivieren
4. Rechtsklick auf Server-Icon (https://discord.gg/UGkgpRFj)
5. "Server-ID kopieren" klicken
6. Das ist deine guildId!

### Methode 2: Aus der URL
- Öffne einen Channel im Server
- URL: `https://discord.com/channels/GUILD_ID/CHANNEL_ID`
- Kopiere die erste Zahl (GUILD_ID)
