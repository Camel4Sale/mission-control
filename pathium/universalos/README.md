# UniversalOS by Pathium 🚀

**Das intelligente Betriebssystem für jede Branche**

---

## 🎯 Vision

UniversalOS ist eine **multi-tenant, branchenagnostische Plattform** die sich dynamisch an jedes Unternehmen anpasst - vom Elektriker bis zum Hotelbetrieb.

---

## ✨ Features

### Core Modules
- 📊 **Dashboard** - Live-KPIs, Umsatz, Team-Auslastung
- 📋 **Auftragsmanagement** - Kanban-Board mit Drag & Drop
- 👥 **Kunden (CRM)** - Vollständige Kundenverwaltung
- 👷 **Team/HR** - Mitarbeiter, Auslastung, Planung
- 💶 **Finanzen** - GuV, Prognose, Liquidität
- 📝 **Angebote** - Erstellung & Tracking
- 🧾 **Rechnungen** - Automatisierter Versand & Mahnwesen
- ⚡ **Automationen** - Workflow-Engine
- 🗺️ **Live-Karte** - Fahrzeug-Tracking, Routenoptimierung

### Platform Features
- 🔌 **Integrations-Hub** - 40+ APIs (DATEV, Slack, Stripe, etc.)
- 🤖 **KI-Marktplatz** - 20+ AI Tools (Claude, GPT, Gemini, etc.)
- 📈 **Benchmarks** - Branchenvergleich
- ⚔️ **Vergleich** - Wettbewerbsanalyse
- 💎 **Preise** - Transparente Pakete
- 🧮 **ROI-Rechner** - Einsparungen berechnen

## 🎨 Design-System

### Typography
- **Display:** Syne (Bold, Character-stark)
- **Body:** DM Sans (Clean, Lesbar)
- **Mono:** JetBrains Mono (Code, Daten)

### Colors
- **Primary:** Industry-dynamisch (configurable)
- **Dark Mode:** #09090D Base
- **Light Mode:** #F4F5F8 Base
- **Accents:** Orange, Blue, Green, Red, Purple, Teal

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/Camel4Sale/pathium-universalos.git
cd pathium-universalos

# Start development server (optional - works standalone)
npx serve .

# Open in browser
open index.html
```

## 🔧 Konfiguration

### Branche ändern
1. Klick auf Branchen-Chip in Topbar
2. Branche wählen (Elektro, Sanitär, etc.)
3. Dashboard passt sich automatisch an

### Branding anpassen
1. Klick auf Firmenname in Sidebar
2. Firmenname, Kürzel, Farbe ändern
3. Theme wählen (Dark/Light)
4. Speichern

### Integrationen verbinden
1. Navigiere zu "Integrations-Hub"
2. Gewünschte Integration wählen
3. API-Key eingeben
4. Verbinden
